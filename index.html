Домашнее задание по GIT
git config --global user.email blabal@bla
Задає електронну пошту користувача
git init
Ініціалізує папку dotGIT в якій зберігаються файли репозиторію, хеші для снапшотів, etc
git status
Показує стан файлів в робочій папці, які файли змінені і не додані або які очікують відправки з буфера(staging area)
git add
Виберає файли, зміни які будуть проведені і перенесені до буфера до часу коміта
git commit
Зберігає всі файли,зміни що ми вибрали з staging area
git log
Перегляд історії комітів
git checkout
Використовується для переключення гілок та вигрузки в робочу папку, etc
git branch
Керування гілками, створення, перегляд, видаленням тощо.
git push
передача файлів "змін" на відділений репозиторій
git pull
заміняє команди git fetch and git merge origin/master - get data from another repo
git merge
злиття гілок
git clone
команда використовується для клонування "копіювання" репозиторіїв

.gitignore - це файл в якому вказуються папки та файли які Gitsdf пропускає і не контролює "не відслідковує" їхні зміни.

git revert
відміняє коміт, відкатує, створює новий коміт без змін
git reset
відміняє зміни
git rebase
обєднує зміни зроблені в одній гільці з іншою гілкою на противагу злиттю
git cherry-pick
забрати коміти з однієї гілки в іншу
git stash
ховає зміни в файлах і зберігає їх окремо, потім можна їх повернути
